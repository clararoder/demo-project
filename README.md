# Wheel of Fortune

A simple, static Wheel of Fortune app built with vanilla HTML, CSS, and JavaScript.

## Features

- 🎡 Interactive spinning wheel with smooth animations
- ➕ Add and remove custom segments via input fields
- 🎲 Random spin duration between 2–10 seconds
- 🏆 Winner announcement on completion

## Demo

Deploy to GitHub Pages by enabling it in your repository settings (source: root of `main` branch).

## Development

Simply open `index.html` in a browser — no build step required.
